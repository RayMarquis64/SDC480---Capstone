To run the application in its current version, I recommend running it from VS Code and using the terminal.

Requirements
	
	Python
	SQLite3
	openpyxl 
	
When in the terminal use
	pip install openpyxl 
	
	then run

	python project_pricer.py 

	The window is expandable, If you cannot see features you are expecting to see, expand the window. 

Run build_windows.bat file to create executable application to Destop

Check desktop for ProjectPricer.exe

Continuing support will include a windows build file that will convert the application into an executable.
